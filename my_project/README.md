# My First Django Project 🌐

Это мой дебютный проект на фреймворке **Django**. Здесь я заложил фундамент знаний о том, как работают современные веб-приложения и как строится архитектура "запрос-ответ".

## 🚀 Что я реализовал

В этом проекте я отработал ключевые механики Django:
* **Маршрутизация (URLs):** Настройка путей, по которым пользователь переходит на разные страницы сайта.
* **Представления (Views):** * Использование `HttpResponse` для отправки простых текстовых ответов.
    * Работа с функцией `render` для динамической генерации страниц.
* **Шаблоны (Templates):** Создание HTML-структуры сайта и передача данных из Python-кода в шаблоны.
* **Архитектура MVT:** Понимание того, как разделяются логика (Views) и визуальная часть (Templates).

## 🛠 Стек технологий
* **Python 3.12**
* **Django 5.x**
* **HTML/CSS** (базовая верстка страниц)
* **Linux Mint** (среда разработки)

## 📦 Как запустить

1. **Клонируйте проект:**
   ```bash
   git clone [https://github.com/main0-zsh/MyFirstDjango.git](https://github.com/main0-zsh/MyFirstDjango.git)
   cd MyFirstDjango
Активируйте окружение и установите Django:

Bash

```bash
./.venv/bin/python3 -m pip install django
```

Запустите локальный сервер:

Bash
```bash
./.venv/bin/python3 manage.py runserver
```
После этого проект будет доступен по адресу: http://127.0.0.1:8000/
